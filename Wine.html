Instalación de Wine

Para ejecutar los capítulos de Poppy Playtime en Linux (Debian 10–13)

Sigue estos pasos en orden. Cada comando debe completarse antes de pasar al siguiente. Si ya tienes Wine instalado, verifica la versión con wine --version.

Paso 1 — Habilitar arquitectura 32 bits

Wine necesita soporte de 32 bits para ejecutar juegos correctamente.

sudo dpkg --add-architecture i386

Paso 2 — Crear directorio para llaves

sudo mkdir -pm755 /etc/apt/keyrings

Paso 3 — Descargar llave de seguridad (WineHQ)

sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key

Paso 4 — Agregar repositorio según tu versión de Debian

Ejecuta solo el comando que corresponda a tu sistema:

Debian 12 / 13 (Bookworm):

sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bookworm/winehq-bookworm.sources

Debian 11 (Bullseye):

sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bullseye/winehq-bullseye.sources

Debian 10 (Buster):

sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/buster/winehq-buster.sources

Paso 5 — Actualizar lista de paquetes

sudo apt update

Paso 6 — Instalar Wine estable

sudo apt install --install-recommends winehq-stable

Paso 7 — Verificar instalación

wine --version
💡 Deberías ver algo como: wine-9.x.x
⚠️ IMPORTANTE: Wine instala soporte para ejecutar los juegos (.exe), no el launcher en sí (que es nativo para Linux). Consulta el manual del launcher para configurar los capítulos en Linux.
