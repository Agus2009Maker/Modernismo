Instalación de Wine (Debian 10 - 13)

Sigue estos pasos para poder ejecutar los capítulos de Poppy Playtime en Linux.

Paso 1: Habilitar 32 bits

sudo dpkg --add-architecture i386

Paso 2: Agregar repositorio oficial (WineHQ)

Descarga la llave de seguridad:

sudo mkdir -pm755 /etc/apt/keyrings
sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key

Paso 3: Selecciona tu versión de Debian

Ejecuta SOLO el comando que corresponda a tu sistema:

Debian 12/13: sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bookworm/winehq-bookworm.sources Debian 11: sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/debian/dists/bullseye/winehq-bullseye.sources

Paso 4: Instalar Wine

sudo apt update
sudo apt install --install-recommends winehq-stable
💡 NOTA: Al finalizar, verifica con wine --version. Recuerda que esto es para los juegos, no para el launcher.
