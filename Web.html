Instalación de Wine (Debian 10-13)

sudo dpkg --add-architecture i386
sudo apt update
sudo apt install wine64 wine32

Cómo usar Playtime Launcher

  1. Descarga el ejecutable desde el botón superior.
  2. Dale permisos de ejecución: chmod +x Playtime.exe.
  3. Ábrelo con Wine y ¡disfruta!
