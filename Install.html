Manual de Instalación

Sigue estos pasos para descargar e instalar el Playtime Launcher correctamente en tu sistema.

1. Descarga

🪟 Windows

Compatible con Windows 10 y Windows 11 (64-bit). Descarga el .rar y extráelo en cualquier carpeta.

DESCARGAR WINDOWS

🐧 Linux

Testado en Debian 10–13 y derivados (Ubuntu, Mint). Descarga el .rar y extráelo. Requiere Wine para ejecutar los capítulos.

DESCARGAR LINUX

2. Instalación en Windows

Paso 1 — Extraer el archivo

Una vez descargado el Playtime_Launcher_Windows.rar, necesitas un extractor de archivos. Se recomienda WinRAR o 7-Zip (ambos gratuitos).

Haz clic derecho sobre el archivo .rar → Extraer aquí (o en la carpeta que prefieras).

Al extraer, el programa te pedirá una contraseña. Introdúcela exactamente así (respetando mayúsculas):

MegaGames

Paso 2 — Ejecutar el Launcher

Dentro de la carpeta extraída, busca el archivo PlaytimeLauncher.exe y haz doble clic para iniciarlo.

En algunos casos Windows puede mostrar una advertencia de SmartScreen. Haz clic en «Más información» → «Ejecutar de todas formas».

💡 No es necesario instalar nada. El launcher funciona de forma portable desde cualquier carpeta.
⚠️ MUY IMPORTANTE — NO MOVER NI MODIFICAR LOS ARCHIVOS

Al extraer el .rar obtendrás varios archivos (generalmente 3 o 4), entre ellos: el .exe del launcher, un .txt con datos necesarios para el funcionamiento, y uno o dos archivos adicionales de configuración.

Todos deben permanecer juntos en la misma carpeta. No muevas el .exe a otra ubicación, no borres el .txt y no modifiques ninguno de los archivos. Si los separas o eliminas alguno, el launcher dejará de funcionar correctamente.

Paso 3 — Tener Steam abierto

El launcher detecta automáticamente tus juegos de Steam. Asegúrate de que Steam esté abierto e iniciado con tu cuenta antes de lanzar cualquier capítulo.

Los capítulos que no poseas en Steam aparecerán bloqueados. Puedes usar versiones personalizadas — ver guía de versiones →

3. Instalación en Linux

Paso 1 — Abrir la Terminal

Todos los comandos de Linux se ejecutan desde la terminal. En Debian podés abrirla con el siguiente atajo de teclado:

Ctrl + Alt + T
💡 Si el atajo no funciona, también podés buscar «Terminal» en el menú de aplicaciones de tu escritorio.

Paso 2 — Extraer el archivo

Usa unrar o un gestor gráfico como File Roller para extraer el .rar. Si no tienes unrar, instálalo con:

sudo apt install unrar
unrar x Playtime_Launcher_Linux.rar

Al extraer te pedirá una contraseña. Introdúcela exactamente así (respetando mayúsculas):

MegaGames
⚠️ MUY IMPORTANTE — NO MOVER NI MODIFICAR LOS ARCHIVOS

Al extraer obtendrás varios archivos (generalmente 3 o 4): el binario del launcher, un .txt con datos necesarios para el funcionamiento, y uno o dos archivos de configuración adicionales.

Todos deben permanecer juntos en la misma carpeta. No muevas el ejecutable por separado, no elimines el .txt y no modifiques ningún archivo. Si los separas, el launcher dejará de funcionar correctamente.

Paso 3 — Dar permisos de ejecución

Navega a la carpeta extraída y da permisos de ejecución al binario del launcher:

chmod +x PlaytimeLauncher

Paso 4 — Ejecutar el Launcher

./PlaytimeLauncher
💡 También puedes hacer doble clic en el archivo desde tu gestor de archivos si ya tiene permisos de ejecución.

Paso 5 — Instalar Wine (para los juegos)

El launcher en Linux es nativo, pero los capítulos del juego son ejecutables de Windows (.exe). Necesitas Wine para ejecutarlos.

Consulta la guía completa de instalación de Wine: Manual de Wine para Linux →

4. Problemas Frecuentes

El launcher no detecta mis juegos de Steam
Asegúrate de que Steam esté abierto e iniciado antes de abrir el launcher. Si el problema persiste, cierra ambos y vuelve a abrirlos en orden: primero Steam, luego el launcher.

Windows bloquea el .exe con SmartScreen
Es normal para software no firmado digitalmente. Haz clic en «Más información» y luego en «Ejecutar de todas formas». El launcher no contiene malware.

El capítulo aparece bloqueado aunque lo tengo en Steam
Verifica que hayas iniciado sesión en Steam con la cuenta correcta. Si el juego está en otra cuenta, usa la opción de Versión Manual apuntando al .exe del juego.

En Linux: el launcher abre pero los juegos no arrancan
Necesitas Wine instalado y configurado. Sigue la guía de instalación de Wine → y luego configura la ruta del .exe en el panel de versiones del launcher.

📋 REQUISITOS MÍNIMOS: Windows 10/11 (64-bit) o Linux con kernel 5.0+. Se recomienda tener Steam instalado para aprovechar la sincronización de logros y progreso en la nube.
